第一步：nodeJS的下载

1.下载网站为 https://kee-hsu.github.io/

2.测试：node -v

第二步：安装HEXO

  1. npm install hexo -g
1.1 root Artasdlkajs


1.2 sudo npm install hero -g    输入密码
  1. 测试hexo

  2. 创建目录并进入

    3.1 hexo init 初始化

    3.2 hexo server 启动本地服务

第三步：you need a github id

  1. 配置SSH-key

    1.1 ssh-keygen 连续按三次回车

    1.2 vim ~/.ssh/id_rsa.pub 复制shh-key

    1.3 github网站设置ssh黏贴

  2. 配置GITHUB与本地机器验证

    1.1 git config - -global user.name “”

    1.2git config - -global user.email “”

  3. 新建博客域名仓库： 用户名.github.io

第四步：config.yml

  1. deploy字段
  2. type:git
  3. repo:域名仓库的HTTPS地址

第五步：单机版Blog可以发布啦

  1. npm install hexo-deployer-git - -save
  2. hexo g
  3. hexo d
