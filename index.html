meta

响应式布局——meta

申明编码

<meta charset='utf-8' />

优先使用 IE 最新版本和 Chrome

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />

浏览器内核控制

<meta name="renderer" content="webkit|ie-comp|ie-stand">  
优先使用webkit内核，上面为优先级顺序

页面关键字

<meta name="keywords" content="your tags，” />

页面描述

<meta name=“description"content="120｜ 150 words" />

搜索引擎索引方式

<meta name=“robots” content="index,follow" /> 
1. all：文件将被检索，且页面上的链接可以被查询；
2. none：文件将不被检索，且页面上的链接不可以被查询；
3. index：文件将被检索；
4. follow：页面上的链接可以被查询；
5. noindex：文件将不被检索；
6. nofollow：页面上的链接不可以被查询。

页面重定向和刷新

<meta http-equiv="refresh" content="0;url=" />  

content内的数字代表时间（秒），既多少时间后刷新，如果加url,则会重定向到指定网页（搜索引擎能够自动检测，也很容易被引擎视作误导而受到惩罚）

网页作者信息

<meta name="author" content="author name" />

CSS单位

CSS3的单位

知识点

  1. em
  2. rem
  3. maxwidth
  4. maxheight
  5. minwidth
  6. minheight
  7. vw
  8. vh

1.em

em是相对父级和自己的字号设置

父级的字号为16px,对自己设置的宽度为 `witth:5em` 则宽度为80px

2.rem

rem是相对于HTML标签的字号设置

html的默认为 `font-size:16px`  最小不能低于12px.

3.maxwidth

最大宽度的意思，该属性值会对元素的宽度设置一个最高限制。因此，元素可以比指定值窄，但不能比其宽。单位可以为像素值，也可以为百分比。像素值定义元素的最大宽度值，百分比定义基于包含它的块级对象的百分比最大宽度。多用于自适应网页。

4.maxheight

为最大高度，基本同maxwidth。

5.minwidth

min-width 属性设置元素的最小宽度。该属性值会对元素的宽度设置一个最小限制。因此，元素可以比指定值宽，但不能比其窄。单位可以为像素值，也可以为百分比。像素值定义元素的最小宽度值，百分比定义定义基于包含它的块级对象的百分比最小宽度。多用于自适应网页
min-width:960px; /* 最小宽度设置为960像素，当浏览器宽度小于该值时，该元素不再自适应，而是部分元素被遮挡住。*/

6.minheight

为最小高度，基本同minheight.

7.vw

为视窗宽度的百分比`img{width:50vw;}`的理解为该图片的宽度为视窗的50%，宽度随视窗的宽度变化。用于自适应网页。

8.vh

类似于vw
